# Sridhar — Portfolio

A professional single‑page portfolio built with HTML, CSS, and vanilla JavaScript.

## Features
- Responsive, accessible layout
- Light/Dark/Auto theme with local preference
- Smooth scrolling and subtle reveal animations
- Project cards, experience timeline, skills
- Contact form with validation and mailto fallback

## Getting started
Just open `index.html` in your browser. No build step required.

## Customization
- Update text content in `index.html`
- Tweak styles in `css/styles.css` (CSS variables at the top)
- Adjust behavior in `js/script.js`
- Replace social/profile links with your own
